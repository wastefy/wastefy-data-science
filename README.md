# Wastefy - Data Science ♻️📊

This repository focuses on data processing, exploratory data analysis (EDA), and machine learning to support an AI-based food inventory management system.

## 📌 Overview
Wastefy aims to reduce food waste by analyzing food characteristics, shelf life, and storage conditions to generate actionable insights.

## 🚀 Setup

1. Clone this repository
```bash
git clone https://github.com/wastefy/wastefy-data-science.git
cd wastefy-data-science
```

2. Create virtual environment
```bash
python -m venv venv
```
3. Activate virtual environment
- Windows
```bash
venv\Scripts\activate.ps1
```
- Mac/Linux
```bash
source venv/bin/activate..ps1
```

4. Install dependencies
```bash
pip install -r requirements.txt
```
